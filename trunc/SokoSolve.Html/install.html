SokoSolve | Installation

The SokoSolve installation process is easy. If you are an experianced user, just click here.

Step 1. Prerequisite: Check .NET 2.0 Runtime (Windows Feature)

SokoSolve requires the Microsoft .NET 2.0 (or better) runtime, which is a standard part of Microsoft Windows. For most people this will already be installed. It is already included in Vista, or is automatically installed via Microsoft Windows Updates. More information about .NET 2.0 redistribution can be found here.

If you are uncertain, do step 2 (download & run installer) -- it will check for you. Otherwise, you can install .NET 2.0 by:

  • Either: via Microsoft Download Center
  • Or: via the Microsoft Windows Update Service, which according to Micrsoft is the preferred method. This only works under Microsoft Internet Explorer.

Step 2. Download the lastest SokoSolve installer

  • Download the installer from the SourceForge download service
    • Click the Download button on this page
    • Select the standard MSI package(*). eg. "SokoSolve-bin-X.X.XX-alpha.msi"
    • Download and then run this file
    • Follow the installer wizard
    • Click "Program Files > Games > SokoSolve Sokoban" to play.

Feedback

If there are any installation problems, or bugs in the software, use the SourceForge bug tracking service to log them. There is also a feature request service, which is great if you have any good ideas or rough edges you would like clean up.

For more information on contributions click here.

Notes

(*) SokoSolve-bin-X.X.XX-alpha.msi -- is the standard installer package. This is for most people.
SokoSolve-bin-debug-X.X.XX-alpha.msi -- is a debug version of the installer which has more helpfull debug symbols, etc.
SokoSolve-bin-xcopy-X.X.XX-alpha.zip -- is a simple xcopy folder distribution. You just unzip and run it.
SokoSolve-src-X.X.XX-alpha.zip -- is the source code snapshot for the release.
-alpha means semi-stable; -beta means stable; -release means final user version.
