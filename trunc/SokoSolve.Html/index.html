SokoSolve | Sokoban Game

SokoSolve - A Sokoban library, game and solver toolset. It is written in C# .NET 2.0 for windows and is GPL open source. SokoSolve is kindly hosted by SourceForge.

SokoSolve is essentially the product of a pet project. But why sokoban?

Status

The current version of SokoSolve is 1.2.0, which represents a complete rewrite to the .NET 2 platform. For a more detailed progress & features report, which is not completely acurate and only pertains to the latest source.

Blocks

Library

An XML library of Sokoban puzzles (XSD, XSLT), were the user can easily manage many puzzle libraries with rich information

Game

A sprite based game UI with little effects. The aim here is to be able to play puzzles from the library. Ths ultimate aim is for the user to play against the solver, or other human players.

Solver

This was the reason this project started. The ability to solver and analyse puzzles from the library. Technically, this is the only block that is interesting. Additional solver information is managed here

Generator

This is essentially the end-goal for the project. To be able to generate interresting (from a human perspective) puzzles, which are checked as valid by the solver, and then played in the Game/Library blocks.
