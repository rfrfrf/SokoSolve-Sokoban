SokoSolve | Sokoban Game

SokoSolve is a Sokoban game implementation and enthusiast tools. Features include: Game, Library, Solver, Editor, and Generator. (Windows, C# .NET 2.0)
SokoSolve is GPL open source, and is kindly hosted by SourceForge.

SokoSolve is essentially the product of a pet project. But why sokoban?

More: Screenshots | Download & Install | SourceForge Project | Puzzle Libraries | Resources | Contributions

Status

The current version of SokoSolve is 1.2.18-alpha release, which represents a complete rewrite to the .NET 2 platform, a brand new architecture and numerous powerful features.

Blocks

Game

Allow the player to solve puzzles in an animated environment, which includes mouse movement. Coming soon sounds effects and music. The ultimate aim is for the user to play against the solver, or other human players.

Library

The real home for any Sokoban enthusiast. The library features an extensible XML implementation with rich content, saved solutions, puzzle alternatives, a puzzle editor, and import and export tools.

Solver

This was the reason this project started. The ability to solver and analyse puzzles from the library. Technically, this is the only block that is interesting. Additional solver information is managed here. As of 1.2.0 the solver has been temporarily removed to focus on stability. It will be reintroduced soon!

Generator

This is essentially the end-goal for the project. To be able to generate interesting (from a human perspective) puzzles, which are checked as valid by the solver, and then played in the Game/Library blocks. This feature is currently under development.
