SokoSolve | Release Notes

Release: 1.3.45-beta

The 1.3.45 is essentially a bug-fix revision to 1.3.44. Thanks to all for your feedback on SokoSolve.

  • Installer did not include all content
  • Redesigned the dynamic resource XML to allow more complete control and skinning of the game block.
  • Solver fixes and cleanups.

Release: 1.3.44-beta

The 1.3 release is the first beta of SokoSolve. It also features the newly rearchitected Solver block. The focus of this release is twofold, to introduce the first cut of the Solver and Stability.

The 1.3.44-beta release includes:

  • Solver Block. See the Solver page
  • Import Wizard (XSB, SLC formats) from file, internet or clipboard
  • Much improved editor
  • Numerous library bug fixes and cleanups

Release: 1.2.18-alpha

The 1.2.18 release is a big step toward a beta version of the Game block. Featuring: Advanced Mouse Movement, Bookmarks, Sound & Music, and a User Profile.

The 1.2.18 release is a big step toward a beta version of the Game block. It features:

  • Advanced Mouse Movement. Player movement (l-click) to any open floor cell. Crate movement (l-click drag) any crate to another open floor cell.
  • Bookmarks. The ability to bookmark a position then restart from it an any further stage
  • Sound & Music. Sound effects and background mp3 playback (this uses embedded Windows Media Player)
  • User Profile. Remember which library and puzzle the user is playing

Release: 1.2.0-alpha

The 1.2.0 is an entirely new codebase and is a completely new architecture and design. It features much more advanced design patterns (perhaps a little over-architected) and is a more mature implementation.

1.2.0-alpha is essentially a pre-release to iron out the kinks in preparation for a mass-appeal beta release. This alpha is for early adopters, contributors and testers.

While 1.2.0-alpha a pre-release, it is intended to be usable and stable.

The 1.2.* release will *NOT* include the solver code and related analysis pieces.
