Release: 1.2.0-alpha

The 1.2.0 is an entirely new codebase and is a completely new architecture and design. It features much more advanced design patterns (perhaps a little over-architected) and is a more mature implementation.

1.2.0-alpha is essentially a pre-release to iron out the kinks in preparation for a mass-appeal beta release. This alpha is for early adopters, contributors and testers.

While 1.2.0-alpha a pre-release, it is intended to be usable and stable.

The 1.2.* release will *NOT* include the solver code and related analysis pieces.
