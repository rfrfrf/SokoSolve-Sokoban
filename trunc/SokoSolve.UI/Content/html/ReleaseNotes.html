SokoSolve | Release Notes

Release: 1.2.17-alpha

The 1.2.17 is a big step toward a beta version of the Game block, featuring: Advanced Mouse Movement, Bookmakes, Sound & Music, and a User Profile.

The 1.2.17 is a big step toward a beta version of the Game block. It features:

  • Advanced Mouse Movement. Player movement (l-click) to any open floor cell. Crate movement (l-click drag) any crate to another open floor cell.
  • Bookmarks. The ability to bookmark a position then restart from it an any further stage
  • Sound & Music. Sound effects and background mp3 playback (this uses embedded Windows Media Player)
  • User Profile. Remember which library and puzzle the user is playing

Release: 1.2.0-alpha

The 1.2.0 is an entirely new codebase and is a completely new architecture and design. It features much more advanced design patterns (perhaps a little over-architected) and is a more mature implementation.

1.2.0-alpha is essentially a pre-release to iron out the kinks in preparation for a mass-appeal beta release. This alpha is for early adopters, contributors and testers.

While 1.2.0-alpha a pre-release, it is intended to be usable and stable.

The 1.2.* release will *NOT* include the solver code and related analysis pieces.
