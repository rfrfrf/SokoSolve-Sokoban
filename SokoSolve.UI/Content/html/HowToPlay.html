SokoSolve | How to play Sokoban

The great thing about Sokoban is that it is really easy to play and intuitive to learn.

Game Rules

You role play as a Japanese warehouse keeper, with the following rules:

  • You can push, but never pull crates
  • You can only push a crate into a free space
  • You win when all the crates are on goal positions
  • You loose when a create gets into a position where it cannot be pushed onto a goal
  • Enjoy.

Just for clarity sake:

  • You cannot push a crate diagonally
  • The order in which you place crate on goals does not matter
  • You can continue playing even though the puzzle is no longer solvable. I.e. If you push a crate into a corner, and that corner is not a goal -- the puzzle cannot to solved from this position; however if you do not notice this you can continue playing in vain.
  • Enjoy.

User Interface

To move the player around you can either:

  • Use the direction key: Up, Down, Left, Right
  • Left-Click on a floor space (you will see little white dots on the floor spaces you can move to)
  • Left-Click on a crate and drag it (again the game will check if you can move the crate)

Additional Commands:

  • U - Undo a step
  • R - Restart the puzzle from the beginning
  • ESC - Give up and return the the main page
