SokoSolve | Sokoban Game

SokoSolve - A Sokoban library, game and solver toolset. SokoSolve is an Sokoban enthusiasts one-stop shop. It is written in C# .NET 2.0 for windows and is GPL open source. SokoSolve is kindly hosted by SourceForge.

The project homepage contains more up-to-date information and new releases.

Have a look at the release notes for more specific information about this version.
