SokoSolve | Feature List

  • For Windows (98, NT, XP, Vista), written in C# .NET 2.0

Game

  • Advanced Mouse Movement. Player movement (L-click) to any open floor cell. Crate movement (L-click and drag) any crate to another open floor cell.
  • Bookmarks. The ability to bookmark a position then restart from it an any further stage
  • Sound & Music. Sound effects and background mp3 playback (this uses embedded Windows Media Player)
  • User Profile. Remember which library and puzzle the user is playing

Library

  • Sort in a Hierarchy of Categories
  • Edit & Create puzzles
  • Store Solutions
  • Store Alternative Puzzles
  • Puzzle hints and learning advice

Solver

Generator
